# 🔐 Login Form

A simple and responsive Login Form built using **HTML**, **CSS**, and **JavaScript**. This project validates user input, provides a user-friendly interface, and redirects to a Welcome page after a successful login.

---

## 📌 Features

- ✅ Responsive design (Mobile & Desktop)
- ✅ Email validation
- ✅ Mobile number validation
- ✅ Password validation (Minimum 8 characters)
- ✅ Show/Hide Password
- ✅ Remember Me checkbox
- ✅ Forgot Password link
- ✅ Login success message
- ✅ Redirect to Welcome page
- ✅ Displays logged-in user's Email/Mobile on the Welcome page
- ✅ Logout functionality

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run the Project

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/Login_Form.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Enter a valid Email or Mobile Number and Password.

5. Click **Login**.

6. After successful validation, you will be redirected to the Welcome page.

---

## ✅ Validation Rules

### Email

- Must be in a valid email format.

Example:

```
example@gmail.com
```

### Mobile Number

- Must contain exactly 10 digits.
- Must start with 6, 7, 8, or 9.

Example:

```
9876543210
```

### Password

- Minimum 8 characters.

Example:

```
Password123
```
